HTML Teglar To‘liq Namoyishi

Quyida siz HTML tilida ishlatiladigan ba’zi maxsus teglarga oid misollarni ko‘rasiz.

1. Gorizontal chiziq <hr>

Bu chiziq sahifa bo‘limlarini ajratishda ishlatiladi:

Bu chiziqdan keyingi matn yangi bo‘limni bildiradi.

2. Ustidan chizilgan matn <strike>

Masalan, eski narxni chizib tashlab, yangi narxni ko‘rsatish uchun ishlatiladi:

Eski narx: 50 000 so‘m — Yangi narx: 35 000 so‘m

3. Oldindan formatlangan matn <pre>

Bu teg matnni qanday yozgan bo‘lsangiz, xuddi shunday chiqaradi:

HTML kod namunasi:


    
        Sahifa
    
    
        Salom, Dunyo!

4. Pastga tushirilgan indeks <sub>

Kimyo formulalarini yozishda juda qulay:

Formulaning misoli: H2SO4

5. Yuqoriga ko‘tarilgan indeks <sup>

Matematika formulalari uchun ishlatiladi:

Misol: x3 + y2 = z2

6. Kod yozuvi <code>

Kod qismini ajratib ko‘rsatish uchun ishlatiladi:

Python kod namuna: print("Salom, Dasturchi!")

JavaScript kodi: console.log("Hello, World!");

7. Kontakt ma’lumotlari <address>

Sayt muallifining manzili yoki aloqasi uchun:

Muallif: Janob
Email: janob@example.com
Manzil: Toshkent, O‘zbekiston
Telefon: +998 90 123-45-67

8. Kommentariya <!-- -->
