HTML class atributi

Class bu HTML elementga beriladigan nom bo‘lib, CSS yoki JavaScript orqali bu elementni boshqarish imkonini beradi.

1. Class atributi sintaksisi

Quyidagi misolda paragrafga "qizil" nomli class berilgan:

div class="code"> <p class="qizil">Bu matn qizil bo‘ladi</p>
class qizil

2. CSS bilan ishlatish

CSS da classga murojaat qilish uchun nuqta (.) belgisi ishlatiladi:

.qizil { color: red; }

3. Bir nechta elementga bir xil class

Quyida ikkita elementga bir xil class berilgan:

class="code"> <h2 class="qizil">Sarlavha</h2>
<p class="qizil">Bu ham qizil matn</p>

4. Bitta elementga bir nechta class

HTML element bir nechta classga ega bo‘lishi mumkin:

<p class="qizil katta">Bu katta va qizil matn</p>

5. class va id farqi

Xususiyat class ID
Qayta ishlatilishi Ha, bir nechta elementda Yo‘q, faqat bitta elementda
CSS belgisi .classNomi #idNomi
JavaScript bilan ishlatish ('.classNomi ') document.getElementById('idNomi')

6. Yakuniy misol (HTML + CSS)

<style>
 .qizil { color: red; }
 .katta { font-size: 24px; }
</style>

<p class="qizil katta">Bu katta va qizil matn</p>
